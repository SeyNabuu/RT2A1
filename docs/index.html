First_node Module

Description:

This node implements an action client to move the robot to a specified goal position. The main function initializes the ros node,set up the ros publisher and subscriber,and starts the action client.

Subscribes to:

  • /odom: Subscribes to the Odometry topic to get the current position and orientation of the robot.

Publishes to:

  • /posevelocity: Publishes the current position, velocity, and distance to the goal of the robot.

Services:

  • /goal_data: Receives a request for the number of goals reached and cancelled..

Second_node Module

Description:

This module implements a service node that provides information about the number of goals reached and cancelled. When called, the node sends a request to the /goal_datà’ service and prints the received information.

Services:

-/goal_data: A service that provides the number of goals reached and cancelled.

Third_node Module

Description:

This node subscribes to the robot’s position, velocity, and distance to the target from the ‘/posevelocity’ topic. It prints this information at a specified frequency.

Subscribes to:

  • /posevelocity: Subscribes to the custom message topic to get the robots position, velocity, and distance from the target.

script.third_node.Callback(msg)[source]

A callback function that subscribes to the posevelocity topic to get the robot’s position, velocity, and distance from target.

Parameters:

msg – the robot’s position, velocity, and distance from target.
